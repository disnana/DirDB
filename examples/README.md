# DirDB Examples

| Language | Example | Run |
| --- | --- | --- |
| Python | [async_basic.py](python/async_basic.py) | `python examples/python/async_basic.py` |
| Rust | [basic](rust/basic) | `cargo run --manifest-path examples/rust/basic/Cargo.toml` |

The Python example uses the async-first public API. The Rust example uses `dirdb-core` directly. Both write disposable data beneath `example-state/`.
